# Volume
Volume component is a storage class manager for [Containerum](https://github.com/containerum/containerum).

## Prerequisites
* Kubernetes

## Installation

### Using Helm

```
  helm repo add containerum https://charts.containerum.io
  helm repo update
  helm install containerum/volume
```
